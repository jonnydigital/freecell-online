## Recommended Strategy

### Option A: "solitairestack.com" — The Brand Play
- Memorable, unique, brandable
- .com authority
- Works for all games: solitairestack.com/klondike, solitairestack.com/freecell
- Easy to say out loud, easy to remember
- **Cost: ~$10/year**

### Option B: "solitaire.games" — The Perfect Name
- Literally "solitaire.games" — doesn't get better than that
- Short, clean, descriptive
- Perfect for a game hub
- .games TLD is less established for SEO but growing
- **Cost: ~$20-30/year**

### Option C: Multi-Domain Network (Long-term)
Keep playfreecellonline.com for FreeCell + register hub domain for everything else:
- **playfreecellonline.com** → FreeCell (existing SEO value)
- **[hub domain]** → Homepage + Klondike + Spider + all variants
- Cross-link between them for authority
- This is what online-solitaire.com does with playonlinesolitaire.com

### Option D: Subdomain Strategy (Free)
Use what we have:
- **playfreecellonline.com** → FreeCell
- **klondike.playfreecellonline.com** → Klondike (weird but works)
- Not recommended — looks amateurish

---

## My Recommendation

**"solitairestack.com" has been purchased as the hub domain.** It's brandable, memorable, and flexible enough for the full network.

Keep playfreecellonline.com as a dedicated FreeCell property that links to the hub.

Later, if revenue supports it, pursue a premium domain like playsolitaire.com or onlinesolitaire.com.

**Also grab "solitaire.games" ($20-30) as a defensive/redirect play.** It's too good a name to let someone else take.
