### Mobile (CSS Code Review — 375px / 390px / 414px)
Browser resize to mobile dimensions was not effective (Chrome minimum window size limitation). Mobile QA conducted via CSS code review and arithmetic verification:
- **Card sizing at 375px**: `clamp(30px, calc((100vw - 61px) / 8), 54px)` → (375-61)/8 = 39.25px ✓ (within 30-54px range)
- **Spider cards at 375px**: `clamp(24px, calc((100vw - 66px) / 10), 44px)` → (375-66)/10 = 30.9px ✓
- **Card sizing at 414px**: (414-61)/8 = 44.125px ✓
- Sidebar: hidden on mobile via `hidden lg:block` (only shows ≥1180px) ✓
- Top bar: `hidden md:flex` — hidden on mobile, visible on tablet+ ✓
- Bottom bar: `flex md:hidden` — visible on mobile with safe-area padding ✓
- Touch handling: `touch-action: none` on cards, `overscroll-behavior: none` on body ✓
- iOS notch support: `viewport-fit=cover` meta tag + `env(safe-area-inset-bottom)` padding ✓

### Tablet (768px–1023px)
