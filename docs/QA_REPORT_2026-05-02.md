### Issue found + fixed

**solitairestack.com homepage meta description was 195 chars** — past Google's typical SERP truncation point (~155–160). The over-long copy was identical in both the Next.js `metadata.description` (line 20 of `src/app/(main)/page.tsx`) and the JSON-LD `HUB_DESCRIPTION` constant (line 141), so SERP and structured data both got the over-long version.

**Fix (commit `401b458`)**: trimmed to 141 chars while preserving the leading FreeCell / Spider Solitaire / Klondike keyword order and the value-prop trio (strategy guides, daily challenges, leaderboards). Updated both occurrences in the same commit so they stay synced.

```
Old (195 chars):
  Play FreeCell, Spider Solitaire, Klondike, and 25+ solitaire card games
  online for free. No download, no signup -- just classic card games with
  strategy guides, daily challenges, and leaderboards.

New (141 chars):
  Play FreeCell, Spider Solitaire, Klondike, and 25 more solitaire
  variants -- free, no signup. Strategy guides, daily challenges,
  leaderboards.
```

`tsc --noEmit` clean. `catalog-truthiness.test.ts` 16/16 PASS. Deploys `dpl_2pu55gjd2aZ3HWiK6Hr18hLLz4AR` (solitairestack) and `dpl_AYYmYypLPLQkYaRxocsqpYujDa9S` (freecell-online) both READY. Live curl verification on `solitairestack.com/` confirms the 141-char meta description and the matching 141-char description in WebSite + WebApplication JSON-LD blocks.
