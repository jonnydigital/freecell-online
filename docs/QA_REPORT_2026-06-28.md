True mobile/tablet were therefore measured via the **same-origin iframe rig** (an iframe genuinely establishes its own layout viewport for its document).

- `solitairestack.com/tripeaks` — renders via the generic cascade board (rows-as-columns), not a canonical 3-peak pyramid. See backlog.
