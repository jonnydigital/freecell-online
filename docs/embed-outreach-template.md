## Customization Notes

- Replace `[site name]` and `[name]` with the actual target
- Replace `[specific reason]` with something concrete about their site — never send a generic pitch
- If the target is a game directory, emphasize the no-ads-in-embed angle
- If the target is educational, mention that FreeCell is a logic/strategy game with no gambling element
- If the target is a forum, suggest it for a pinned games thread or sidebar widget
- Keep the email under 150 words total — shorter is better

## When Not to Use This Template

- If the site has no obvious place for an embedded game
- If the site does not accept external content or embeds
- If the contact is a generic info@ address with no editorial presence
