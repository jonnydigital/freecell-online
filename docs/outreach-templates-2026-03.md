## Template Rules

- Never send more than one follow-up
- Always personalize — generic pitches get ignored
- Keep every email under 150 words
- Never misrepresent what the asset is (e.g., don't call the embed "a WordPress plugin")
- If there's no clear editorial fit, don't send the pitch
- Track every send in `docs/outreach-wave-1-tracking.md`
